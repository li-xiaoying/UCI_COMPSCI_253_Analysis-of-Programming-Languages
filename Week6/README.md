# Compile and Run the program with the following commands

## Week 6

```
cd Week6
```

### TwentyNine
To compile TwentyNine.cpp in Shell:
```
clang++ -std=c++20 -pthread TwentyNine.cpp -o TwentyNine
```
To run TwentyNine in Shell:
```
./TwentyNine ../pride-and-prejudice.txt
```
